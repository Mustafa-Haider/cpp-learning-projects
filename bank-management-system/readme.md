# C++ Bank Management System

A console-based Bank Management System built while learning core C++ programming fundamentals. This project simulates basic banking operations such as account creation, PIN authentication, deposits, withdrawals, balance checks, account details, and transaction history using functions, loops, switch-case, and validation logic.

## Features
- Create Account with age and minimum deposit validation
- 4-digit PIN setup and authentication
- Deposit Money
- Withdraw Money with minimum balance protection
- Check Balance
- View Account Details
- Basic Transaction History (last deposit and withdrawal)
- Total transaction count
- Menu-driven interface
- Continuous operation until user exits

## Concepts Practiced
- Functions
- Global variables
- Switch-case statements
- While loops
- Conditional logic
- User authentication
- Input validation
- State management

## Technologies Used
- C++
- Standard Library (`iostream`, `string`)
